# PDF Resumer com IA

Aplicação web que resume PDFs automaticamente usando inteligência artificial.

## 🚀 Tecnologias
- Python
- Streamlit
- Groq API

## 📌 Funcionalidades
- Upload de PDF
- Extração de texto
- Resumo automático em português

## ▶️ Como rodar

"""
git clone https://github.com/joaovitorsoska/bot-pdf-resumer.git
cd bot-pdf-resumer

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
streamlit run app.py
"""

## 🔑 Configuração

Crie um arquivo `.env` na raiz do projeto com:

GROQ_API_KEY=sua_chave_aqui
