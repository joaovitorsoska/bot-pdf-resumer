Bot PDF Resumer
Este projeto é um chatbot em Python que permite resumir arquivos PDF usando Inteligência Artificial.
A ideia surgiu como forma de praticar a leitura de PDFs, integração com modelos de linguagem e construção de um chatbot simples via terminal.
Mesmo que o PDF esteja em outro idioma, o resumo final é sempre gerado em português.

* O projeto lê arquivos em PDF
* Envia o conteúdo para um modelo de IA
* Retorna um resumo direto e objetivo
* Funciona via Terminal
* Usa um comando simples para resumir PDFs

O comando usado para para resumir é, por exemplo: "resumir artigo.pdf"

Para fazer este projeto usei Python, PyPDF2 (Leitura do PDF), LangChain, OpenRouter (modelo llaMA 3.1), NLTK, Git e Github.


O POR QUE FIZ ESSE PROJETO?
O propósito do projeto é por em prática oque estava estudando e aprendendo:
*  Como trabalhar com arquivos PDF em Python
*  Como integrar IA em um projeto real
*  Estrutura chatbot baseado em comandos

Como rodar:
* Clone o repositório
* Crie um ambiente virtual
* instale as dependências
* Execute o arquivo "chatbot.py"

Algumas coisas vão ser feitas ao longo do tempo no projeto! A ideia é continuar aprimorando o bot conforme as aprendizagens forem avançando
